# Demo application of the use of Yew Router in the Yew.rs framework + PWA.

## To run the app:

Add the WebAssembly target::

```
$ rustup target add wasm32-unknown-unknown
```

Install Trunk to run the app:

```
$ cargo install trunk
```

Run the app:

```
$ trunk serve
```

Build the application by running:

```
$ trunk build --release
```
